# Delete-Exif
Delete a Exif at JPEG file.

### Using a library

* PIL

------

### How to use

1. Put a target file (jpeg format only) in the "work" directory.
2. Run to python script!
3. Image files(Erased a exif data) stored a "result" directory by Python script.
